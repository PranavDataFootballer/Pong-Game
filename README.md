# Pong-Game
A classic Pong game built with Python, featuring smooth paddle controls, ball physics, collision detection, score tracking, and simple arcade-style gameplay.

A classic **Pong Game** developed using **Python**. This project recreates the iconic arcade game where two players compete by bouncing a ball back and forth using paddles. It is a great beginner-friendly project for learning game development concepts such as collision detection, game loops, keyboard input, and basic physics.

---

## Features

- Two-player gameplay
- Smooth paddle movement
- Ball collision detection
- Score tracking
- Ball speed increases after collisions 
- Simple and clean user interface
- Lightweight and easy to run

---

## Technologies Used

- Python 3.x
- Turtle Graphics *(or Pygame if applicable)*
- Object-Oriented Programming (OOP)

## Controls

| Player | Controls |
|---------|----------|
| Left Paddle | W (Up), S (Down) |
| Right Paddle | ↑ (Up), ↓ (Down) |

---

## Gameplay

- Prevent the ball from passing your paddle.
- Each missed ball awards one point to the opponent.
- The player with the highest score wins.
